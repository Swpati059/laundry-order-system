# 🧺 Laundry Order Management System (AI-First)

A full-stack web app using Flask and JavaScript to manage laundry orders.

---

## 🚀 Features

- Create orders
- Calculate billing
- Track status (RECEIVED → PROCESSING → READY → DELIVERED)
- View orders dashboard

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JS
- Backend: Flask
- API: REST
- CORS enabled

---

## 📁 Project Structure

laundry-system/
├── backend/
├── frontend/
├── README.md

---

## ⚙️ Run Project

cd backend
pip install -r requirements.txt
python app.py

Open:frontend/index.html

## ⚖️ Tradeoffs

Due to limited time (72 hours), I focused on building a working and functional system.

- Used in-memory storage instead of a database (MongoDB/SQL)
- Did not implement authentication (login/signup)
- Used a basic frontend (HTML/CSS) instead of advanced frameworks like React

### What I would improve with more time:

- Add database integration (MongoDB)
- Implement user authentication
- Add search and filtering features
- Deploy the application online
